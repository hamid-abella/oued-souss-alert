# Oued-Souss Alert - Backend

## Prérequis

- Node.js ≥ 18
- PostgreSQL ≥ 14

---

### Configurer le Backend

cd ../backend

npm install

Créer fichier .env :

PORT=5000\
DB_HOST=localhost\
DB_PORT=5432\
DB_USER=postgres\
DB_PASSWORD=your_password\
DB_NAME=oued_souss_db

### Lancer le Backend

npm run dev

Tester :
http://localhost:5000/api/zones
